# Leon - Creative Digital Agency Template

![Leon Template Screenshot](images/main-section.jpg)

## Description
Leon is a minimal and responsive web template for creative digital agencies.  
It includes sections for landing, features, services, portfolio, about, and contact. Built using **Bootstrap 5** and **Font Awesome**.

## Features
- Fully responsive design  
- Landing Page with Hero section  
- Services showcase  
- Portfolio gallery  
- About section  
- Contact section with mailto link and social icons  
- Built with Bootstrap 5 and Font Awesome  

## Technologies
- HTML5 & CSS3  
- Bootstrap 5  
- Font Awesome  
- Google Fonts (Work Sans)
